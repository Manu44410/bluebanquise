BlueBanquise Documentation
==========================

.. image:: images/BlueBanquise_logo.svg
   :align: center

------------

This documentation contains information on how to deploy and use the BlueBanquise stack
or BlueBanquise Ansible collection.

This documentation also contains few tutorials indirectly related to BlueBanquise, for generic knowledge.

Users can use left navigation bar to navigate to desired part of this documentation,
or scroll down this page to see the whole documentation summary.

`Please report us <https://github.com/bluebanquise/bluebanquise/issues>`_ any
issues in this documentation.
